The generated ltoken value for the user.

To generate this run: 'validate auth --url http://localhost:8080 --insecure', note that insecure is required to create a portable token that is also usable with the API.

Afterwards run: 'validate auth --url http://localhost:8080 --print-auth-info', which should return something like this:
{"username":"jsmith","token":"39dbca010c2d753c68dd39252598dced09c290b7e66cb6a8fe995bbe86011290"}
Copy the token value i.e 39dbca010c2d753c68dd39252598dced09c290b7e66cb6a8fe995bbe86011290 and enter it here.
